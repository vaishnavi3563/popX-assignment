# PopX React Assignment

This project is a mobile-first React application built as part of a
qualifier assignment.

## Features
- Pixel-perfect UI based on Adobe XD design
- React Router navigation
- Signup & Login with validation
- Dynamic Account page
- Context API + localStorage
- Bottom navigation with Home and < > controls

## Tech Stack
- React
- Vite
- React Router
- Context API
- CSS
